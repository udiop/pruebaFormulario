What is Api2Pdf.com?

Api2Pdf.com is a REST API for instantly generating PDF documents from HTML, URLs, Microsoft Office Documents (Word, Excel, Powerpoint), and images. The API also supports merge / concatenation of two or more PDFs. Api2Pdf is a wrapper for popular libraries such as wkhtmltopdf, Headless Chrome, and LibreOffice. You can also generate barcodes and QR codes with Zebra Crossing (ZXING).

Acquire API Key

Create an account at portal.api2pdf.com to get your API key. It only takes a minute.

Test the API

From the URL below, replace {YourApiKey} with your API key you acquired and then copy and paste the URL below into a new browser window to test a simple GET request.

https://v2018.api2pdf.com/chrome/url?url=https://www.github.com&apikey={YourApiKey}
